 Vue后台管理通用解决方案

## 使用

```bash
$ git clone https://github.com/xinpingJun/vue-admin.git
$ cd vue-admin
$ yarn install
$ yarn serve         # 访问 http://localhost:8000
```

## 简介
&emsp;&emsp;Iview Pro是基于Vue.js，搭配使用[iView] ，使用vue-cli 3.0 脚手架搭建的 Vue后台管理通用解决方案。

## 项目结构
```shell
.
└── src
    ├── api  接口文件
    ├── assets  静态文件
    ├── libs  工具方法
    ├── router  路由配置
    ├── store  状态管理
    ├── vendors  公共库文件
    └── views
        ├── login  登陆页面
        ├── main  主框架
```
