## 使用

> 安装依赖 `yarn install`

> 运行项目 `yarn serve`

> 打包项目 `yarn build`


## 简介
&emsp;&emsp;i-admin是基于Vue.js，搭配使用[iView] ，使用vue-cli 3.0 脚手架搭建

 
## 项目结构
```shell
.
└── src
    ├── api  接口文件
    ├── assets  静态文件
    ├── libs  工具方法
    ├── router  路由配置
    ├── store  状态管理
    ├── vendors  公共库文件
    └── views
        ├── login  登陆页面
        ├── access  权限管理
        ├── error_page  错误页面
        ├── form  表单编辑
        ├── home  首页
        ├── international  多语言
        ├── message  消息中心
        ├── my_components  业务组件
        └── tables  综合表格
```
